# AI-Редактор комментариев (GAS)

Скрипт для Google Apps Script добавляет меню `AI-Редактор` в Google Таблицу и по строкам листа `Контент` генерирует уникальный редакторский комментарий в столбец `K` через API (OpenRouter/ProxyAPI).

## Структура таблицы

Лист `Контент`:
- `A1` — информация о фильме (используется как часть контекста)
- `C` — название канала/паблика (БОРЩ, Mash и т.д.)
- `D` — рекламная метка (`есть/нет`, `true/false` и т.п.)
- `E` — дата выхода поста
- `G` — исходный текст поста
- `K` — вывод сгенерированного комментария

Лист `Settings`:
- `A2` — API Key
- `B2` — **базовый** URL API (скрипт сам допишет `/chat/completions`, если нужно). Примеры:
  - ProxyAPI: `https://api.proxyapi.ru/openai/v1`
  - **Polza.ai** (OpenAI-совместимый): `https://polza.ai/api/v1` — полный метод: `POST …/v1/chat/completions` ([документация](https://polza.ai/docs/gaidy/chat-completions))
- `C2` — ID модели в формате провайдера (например, `gpt-5-nano` или `deepseek/deepseek-v3.2-speciale` для Polza)
- `E2:E` — список «Золотых правил» (каждое правило в отдельной ячейке)

**Polza.ai:** в `B2` достаточно указать `https://polza.ai/api/v1` или `https://polza.ai/api` — не нужно вручную добавлять `/chat/completions` (скрипт добавит сам). Ключ — в `A2` как `Bearer` (как в примерах Polza).

## Как использовать

1. Добавьте файл `Code.gs` в проект Google Apps Script, привязанный к вашей таблице.
2. Сохраните проект и откройте таблицу заново: появится меню `AI-Редактор`.
3. Выделите ячейку на нужной строке (или диапазон строк) на листе `Контент` и выберите:
   - `Обработать текущую строку`
   - `Обработать выделенный диапазон`

4. Окно **«Дополнить промпт»** (опционально): в `Code.gs` константа `ENABLE_EXTRA_PROMPT_DIALOG` — при `false` диалог не показывается (удобно для тестов). Если включено: можно ввести пожелания или оставить пустым; **Отмена** — прерывает запуск без вызова API.

Во время обработки будет показываться toast `AI думает...`.

Лист **`Logs`**: новые записи добавляются **сверху** (под заголовок). Для провайдеров вроде **Polza.ai** в ответе приходит `usage` со стоимостью в рублях — в лог пишется строка **`USAGE`** с колонками **`cost_rub`**, **`total_tokens`** и полным JSON в `details` ([учёт средств Polza](https://polza.ai/docs/osobennosti/usage)).
