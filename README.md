# AI-Редактор комментариев (GAS)

Скрипт для Google Apps Script добавляет меню `AI-Редактор` в Google Таблицу и по строкам листа `Main` генерирует уникальный редакторский комментарий в столбец `K` через API (OpenRouter/ProxyAPI).

## Структура таблицы

Лист `Контент`:
- `A1` — информация о фильме (используется как часть контекста)
- `C` — название канала/паблика (БОРЩ, Mash и т.д.)
- `D` — рекламная метка (`есть/нет`, `true/false` и т.п.)
- `E` — дата выхода поста
- `G` — исходный текст поста
- `K` — вывод сгенерированного комментария

Лист `Settings`:
- `A2` — API Key
- `B2` — API URL (например, `https://api.proxyapi.ru/openai/v1`)
- `C2` — модель (например, `gpt-4o`)
- `E2:E` — список «Золотых правил» (каждое правило в отдельной ячейке)

## Как использовать

1. Добавьте файл `Code.gs` в проект Google Apps Script, привязанный к вашей таблице.
2. Сохраните проект и откройте таблицу заново: появится меню `AI-Редактор`.
3. Выделите ячейку на нужной строке (или диапазон строк) на листе `Main` и выберите:
   - `Обработать текущую строку`
   - `Обработать выделенный диапазон`

Во время обработки будет показываться toast `AI думает...`.
